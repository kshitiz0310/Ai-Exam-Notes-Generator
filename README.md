## 📌 About The Project

**AI Exam Notes Generator** is built to solve one of the most common student problems — spending hours making notes before exams. Just feed in your topic or content, and get clean, structured, easy-to-revise notes powered by AI.

> Because last-night-before-exam panic deserves a shortcut. 😅

---

## ✨ Features

- 🤖 **AI-Powered Note Generation** — Converts raw content/topics into structured notes
- ⚡ **Instant Results** — No waiting, no manual formatting
- 🔐 **Secure Authentication** — Login/Signup with protected routes
- 📄 **Clean, Readable Output** — Notes formatted for quick revision
- 📱 **Responsive Design** — Works seamlessly across devices
- 🌐 **Full-Stack Architecture** — Separate client & server for scalability

## ⚙️ Getting Started (Local Setup)

```bash
# Clone the repo
git clone https://github.com/kshitiz0310/Ai-Exam-Notes-Generator.git

# Go into the project folder
cd Ai-Exam-Notes-Generator

# Install dependencies (frontend)
cd client
npm install

# Install dependencies (backend)
cd ../server
npm install

# Run backend
npm run dev

# Run frontend (in a new terminal)
cd ../client
npm run dev
```

Create a `.env` file in the `server` folder with:

```env
MONGO_URI=your_mongodb_atlas_uri
PORT=5000
JWT_SECRET=your_secret_key
AI_API_KEY=your_ai_api_key
```
